# Zenova - Modern SaaS Homepage

A fully responsive, modern, and visually appealing **SaaS homepage** built with **HTML, Tailwind CSS, and custom CSS**, without any JavaScript. Zenova is designed as a clean, interactive landing page for SaaS platforms, featuring a hero section, features, pricing, testimonials, call-to-action, and a modern footer.

---

## Features

- **Hero Section**: Full-screen gradient background with floating decorative shapes and clear CTA buttons.
- **Responsive Design**: Fully responsive layout for mobile, tablet, and desktop using Tailwind CSS grids and utilities.
- **Features Section**: Highlight key product features with cards and hover effects.
- **Pricing Section**: Show flexible pricing plans with interactive buttons and highlighted Pro plan.
- **Testimonials Section**: Showcase customer feedback in a clean grid layout.
- **Call-to-Action (CTA)**: Prominent section encouraging users to sign up.
- **Modern Footer**: Includes brand info, quick links, social media, and newsletter subscription form.
- **Interactive Effects**: Hover animations, floating shapes, and subtle transitions using CSS only.
- **No JavaScript**: Entire page uses **HTML, Tailwind CSS, and CSS**, ensuring fast load times and simplicity.

---

## Tech Stack

- **HTML5** – Structure and semantic elements.
- **Tailwind CSS** – Utility-first framework for styling and responsive design.
- **Custom CSS** – For animations, floating shapes, and hover effects.
